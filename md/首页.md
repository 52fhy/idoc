
通过markdown生成静态页面的工具。经常用 markdown 写一些接口文档，为了方便特写此工具玩耍。[生成的页面预览效果](http://jaywcjlove.github.io/idoc) 

```
    ,,        ,,
    db      `7MM
              MM
  `7MM   ,M""bMM  ,pW"Wq.   ,p6"bo
    MM ,AP    MM 6W'   `Wb 6M'  OO
    MM 8MI    MM 8M     M8 8M
    MM `Mb    MM YA.   ,A9 YM.    ,
  .JMML.`Wbmd"MML.`Ybmd9'   YMbmd'
```

# 全局安装

```bash
$ npm install idoc -g
```
